Top 5 free sources to host your website

Hosting

Hosting is a service provided by many companies which provides space on a web server for our website to store its files and other information. You can host your site using any platform such as github, Hostinger, bluehost, Godaddy etc. In hosting plans they provide free space for a website, free domain and ssl certificate, email support etc. So that you can manage your website effectively.

Top 5 free sources to host your website

Github

Github is a popular platform used for collaboration on software projects. You can make your developer account there and share your projects, make repositries, pull request and learn various programming languages. It's also the best source to host nodejs applications and static websites.

Netlify

Netlify is a free open source to deploy your projects. It provides 100gb free disk space, sub domain and ssl certificate for your website. In my opinion it is best platform to host your any static website. You just need to drag and drop your files here and it automatically deploy your site.

You can also add your custom domain in it and it automatically secured your site by https.

Infinityfree

Infinityfree provides good web hosting with no expiration date, It provides many services with free of cost include unlimited storage, free subdomin, cpanel where you get easy management of emails, secure database and other installation options. It is suitable to host php websites, it's disadvantage is that there are no support options available to contact them and it is not suitable for big websites.

Firebase

Firebase provides fast and secure hosting, you can also built many web appliactions through it by writing few commands. It free provides ssl certificates with each custom domain. In terms of services and security firebase hosting is better.

Heroku

Heroku provides free web hosting for developers to learn and built their websites on their platform. It is the best platform to host full stacked websites inculding backend and database using Nodejs and mongodb with just few command line code. So if you want to host nodejs applications heroku is the best
