How to create simple form validation using Javascript

Before submitting a form there is need to validate it by spotting errors to aviod the submission of incorrect information entered by the user. So with the help of Javascript we can show error to a user if he/she entered the wrong information while submitting a form and also display a message when the form is successfully submitted.

So here i create a simple form validation using Javascript by selecting the elements of the form such as password,name,email. And made two functions for success and error, you can also add buttons corresponding to each success and error. Here is the code through which you can understand it.
