How to create an animated game using Javascript

There are various kinds of events in Javascript which are used for different purposes such as keyboard events, mouse events, form events, CSS events etc, these events are used to make websites interactive when a user performs any action.

How to made animations for a game?

The game has three characters first is player and the other two are obstacles. Using CSS we can made awesome animations such as jumping, running of a game character. When user presses a key in keyboard we can set these animations accordingly. There are keydown events in Javascript which helps us to move an object when user presses a particular keyboard key. Here is the code through which you can uderstand the concept, here class dino is the class given to our main character which we want to move, animate class is used to jump a character. 


document.addEventListener('keydown', (e) => {
console.log("key is" + e.key);


if (e.key == "ArrowUp") {
dino = document.querySelector(".dino")
dino.classList.add("animate");
audio.play()
setTimeout(() => {
dino.classList.remove("animate")
}, 700);
}

else if (e.key == "ArrowRight") {
dino = document.querySelector(".dino")
dinox = parseInt(window.getComputedStyle(dino, null).getPropertyValue("left"))
dino.style.left = dinox + 60 + "px"

}

else if (e.key == "ArrowLeft") {
dino = document.querySelector(".dino")
dinox = parseInt(window.getComputedStyle(dino, null).getPropertyValue("left"))
dino.style.left = dinox - 60 + "px"
}


})

We can set collision between obstacle and player as follows:
First we need the x and y length of a character and obstacle we can do this by using window.getcomputedstyle, and then calculate the the difference between their x and y length. We can also set another animation name dead and add audio when the game start and end.
